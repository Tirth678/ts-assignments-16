# TypeScript Assignments

This repository contains a set of TypeScript practice exercises organized by difficulty level.

## Structure

- `easy/` - Basic TypeScript problems for learning foundational concepts.
- `medium/` - Intermediate TypeScript problems focusing on unions, interfaces, and type narrowing.

## Included Files

### `easy/`

- `greet.ts` - Simple function and string handling.
- `multipleOf.ts` - Numeric checks and conditional logic.
- `multiply.ts` - Multiplication and arithmetic operations.
- `runLater.ts` - Delayed execution or timing logic.

### `medium/`

- `combined.ts` - Working with combined types or intersections.
- `domain.ts` - Domain-specific type handling.
- `driving.ts` - Driving-related logic and conditions.
- `tesla.ts` - Likely a type or interface example for specific brand features.
- `union.ts` - Demonstrates union types and type narrowing for shapes.

## How to Use

1. Open a file from the `easy/` or `medium/` directory.
2. Read the problem statement at the top of the file.
3. Implement or test the TypeScript logic as needed.

## Notes

- The repository is intended for practice and learning.
- Each file is self-contained and includes its own example usage.
- The current implementation is written in TypeScript and can be run with `ts-node` or compiled with `tsc`.
